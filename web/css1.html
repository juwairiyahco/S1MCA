WELCOME TO THE WORLD OF CSS

Properties of CSS
  • A CSS property assign a style or behavior to an HTML element. Examples include: color, border, margin, font-style, and transform.
  • Possible values are left , right , center , and justify (for full justification with to ragged left or right margin).
  • There are more than 100 different properties in CSS and a nearly infinite number of different values. Not all pairs of properties and values are allowed and each property defines what are the valid values.
  • Every CSS declaration includes a property / value pair
